LilGuy Sprite Sheet Guide

This guide explains how to prepare the sprite sheets for the LilGuy pixel art dashboard. The sprites have been implemented in the code but need proper image files to function.

Sprite Sheet Specifications

  • Each sprite is 32x32 pixels
  • Sprite sheets are organized in rows of animations
  • Each row contains different frames of the same animation
  • Multiple colors needed: green, blue, and black
  • Format should be PNG with transparency

Colors Used:

Green: #4CAF50
Blue: #2196F3
Black: #333333

Sprite Sheet Layout

Each sprite sheet follows this row layout for animations:

  • Row 0: Idle animation (6 frames)
  • Row 1: Walk animation (5 frames)
  • Row 2: Happy animation (6 frames)
  • Row 3: Angry animation (5 frames)
  • Row 4: Sad animation (6 frames)
  • Row 5: Shocked animation (6 frames)

For egg sprites:

  • Row 0: Idle animation (6 frames)
  • Row 1: Shake and hatch animations (first 2 frames for shake, last 4 for hatch)

For hatchling sprites:

  • Row 0: Idle animation (4 frames)
  • Row 1: Walk animation (4 frames)
  • Row 2: Happy animation (4 frames)
  • Row 3: Sad animation (4 frames)

Steps to Create the Sprite Sheets

  1. Take the blue and green sprite images you've shown
  2. Split them into the individual sprite sheet files according to the list above
  3. Each file should contain only the animations for its specific character type and color
  4. Make sure all sprite frames are positioned accurately according to the animation row layout
  5. Save each file as a PNG with transparency
  6. Place them in the /public/assets/sprites/ directory

You can use image editing software like Photoshop, GIMP, or Aseprite to create these sprite sheets.

Code Implementation

The code is already set up to use these sprite sheets. The key parts are:

// In SpriteManager.ts
export const getSpriteSheetPath = (stage: LilGuyStage, color: LilGuyColor): string => {
  if (stage === 'egg') {
    return `/assets/sprites/egg_${color}.png`;
  } else if (stage === 'hatchling') {
    return `/assets/sprites/hatchling_${color}.png`;
  } else if (stage === 'angel') {
    return `/assets/sprites/angel_${color}.png`;
  } else if (stage === 'devil') {
    return `/assets/sprites/devil_${color}.png`;
  } else {
    return `/assets/sprites/lilguy_${color}.png`;
  }
};

The animations are set up to extract the correct frames from each sprite sheet based on the character's current state and animation.
